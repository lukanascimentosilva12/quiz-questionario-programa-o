# Quiz Interativo para Programadores

## Sobre

Projeto desenvolvido para praticar HTML e CSS através da criação de um quiz sobre programação.

## Tecnologias

- HTML5
- CSS3

## Funcionalidades

- Perguntas de múltipla escolha
- Campos de texto
- Upload de arquivo
- Tabela de pontuação
- Layout estilizado
